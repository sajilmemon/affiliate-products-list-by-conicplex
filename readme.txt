== Description ==
Affiliate Products List by ConicPlex

Now generate or create an amazing and beautiful Affiliate Products List quickly with the help of this tool, for FREE.

How to Use this Tool?

* After installing and activating the plugin, head-over to Menu >> Affiliate Products List by ConicPlex >> Add New.
* Here you can add a new list of products to be added in your Post. So start with giving a name to your list.
* Add the multiple products you have along with their images, description and landing page link (affiliate link).
* Once done, save the list and you will get the shortcode which you can use in your posts or pages to display the table.
* Go to Posts in which you would like to display the list and there paste the shortcode that you got in the previous step.
* Once done, save the post and you can visit the post page to see how your list of products look.
* You can customize or modify the look and feel of the table or list of products by going to Menu >> Affiliate Products List by ConicPlex >> Settings.

== Installation ==
Activating the Affiliate Products List by ConicPlex plugin is just like any other plugin.

1. In your WordPress admin, go to Plugins > Add New
2. In the Search field type “Affiliate Products List by ConicPlex”
3. Under “Affiliate Products List by ConicPlex” click the Install Now link
4. Once the process is complete, click the Activate Plugin link
5. Now, you’re able to use Affiliate Products List by ConicPlex Plugin
